Project Overview

CampusSwap is a campus sustainability frontend where users connect a wallet, complete a lightweight profile, submit verified green actions, receive GT and RT, redeem rewards, and review governance and verifier flows.

Current stack: Next.js 14, TypeScript, React, Tailwind CSS, RainbowKit, wagmi, viem, and Supabase.

Current wallet mode: MetaMask only.

Collaboration Notes

  • Use the Solidity contracts as the source of truth for behavior, thresholds, and enums.
  • Do not hardcode contract addresses inside page components.
  • Wallet addresses stored in Supabase should always use lowercase.
  • Transaction success should only be shown after a confirmed receipt.
  • MetaMask is the current wallet experience used by the shipped frontend.
Recommended reading order: Routes → Hooks → Shared UI → Contracts and ABIs.
