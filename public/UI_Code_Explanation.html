一、项目总览

CampusSwap 是一个围绕校园可持续行为激励的前端系统。用户通过钱包连接进入平台，完成轻量 profile 后，可以提交绿色行为、等待 verifier 审核、获得 GT / RT、参与兑换，并查看奖励、兑换历史和治理页面。

当前前端采用：Next.js 14、TypeScript、React、Tailwind CSS、RainbowKit + wagmi、viem、Supabase。

当前钱包连接模式：MetaMask only，不再依赖 WalletConnect Project ID。

八、当前已经完成的关键修复

  • 钱包连接已改成 MetaMask-only，不再强依赖 WalletConnect Project ID。
  • Profile 读写已补 app/api/profile/route.ts，前端不再直接通过浏览器 fetch Supabase。
  • 提交页已改为英文日期输入 YYYY-MM-DD，文件上传改为英文按钮和英文状态文本。
  • 首页已增加更完整的介绍内容和 Level 展示，文案更适合校园场景。
  • 已增加环境变量提示横幅，若 .env.local 缺配置会直接提示。

十一、组员快速上手顺序

  1. 先看 app/page.tsx，理解首页结构
  2. 再看 app/submit/page.tsx + hooks/use-submit-task.ts，理解完整核心流程

十二、当前结论

这份前端已经不是静态 demo，而是一个有页面结构、钱包连接、profile gate、合约调用入口、Supabase 读写、上传接口、奖励、治理、审核、兑换模块的多页面前端工程。

如果组员要继续接手，优先从 app/、hooks/、app/api/profile/route.ts、lib/supabase/ 入手即可。
