==================================================
一、项目总览
==================================================

CampusSwap 是一个围绕校园可持续行为激励的前端系统。
用户通过钱包连接进入平台，完成轻量 profile 后，可以：
1. 提交绿色行为并上传证据
2. 等待 verifier 审核
3. 获得 GT / RT 并参与兑换
4. 查看奖励、兑换历史和治理页面

当前前端采用：
- Next.js 14（App Router）
- TypeScript
- React
- Tailwind CSS
- RainbowKit + wagmi
- viem
- Supabase

当前钱包连接模式：
- MetaMask only
- 不再依赖 WalletConnect Project ID
